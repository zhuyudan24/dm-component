```shell
npm install @gic-test/vue-gic-datepicker -S
```

## how to use
```

import vueGicDatepicker from '@gic-test/vue-gic-datepicker'

Vue.use(vueGicDatepicker)

// 使用页面
<vue-gic-datepicker></vue-gic-datepicker> 

```
